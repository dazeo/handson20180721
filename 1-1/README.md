**1-1 WEBページの基本**

WEBページは、HTMLという拡張子のファイルでできている。
HTMLファイルをテキストエディタで表示してみると、本文のほかに<>で囲まれた多数のタグが表れる。
文章の構造や、図やリンクの挿入などは、このタグによって制御されており、WEBブラウザがHTMLファイルを読み込む際に、タグに基づいてページの再構築を行う。
